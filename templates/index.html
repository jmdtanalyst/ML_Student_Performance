Student Exam Performance Indicator

This project aims to predict the performance of students in math exams using various factors, such as writing and reading scores, gender, race or ethnicity, parental level of education, lunch type, and test preparation course. To achieve this, we employed several machine learning models, including random forest, decision tree, gradient boosting, linear regression, XGBoost, CatBoosting, and AdaBoost.

Features Used for Prediction

  • Writing Score
  • Reading Score
  • Gender
  • Race or Ethnicity
  • Parental Level of Education
  • Lunch Type
  • Test Preparation Course

Machine Learning Models Employed

  • Random Forest
  • Decision Tree
  • Gradient Boosting
  • Linear Regression
  • XGBoost
  • CatBoosting Regressor
  • AdaBoost Regressor
